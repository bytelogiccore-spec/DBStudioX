# 빌드 및 배포 방법

## 로컬 개발 환경 설정

### 필수 요구사항

- Node.js 20 이상
- Rust stable (최신 버전)
- Windows 10/11 (개발 환경)

### 설치 단계

1. **저장소 클론**
   ```bash
   git clone https://github.com/ByteLogicCore/DBStudioX.git
   cd DBStudioX
   ```

2. **의존성 설치**
   ```bash
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run tauri:dev
   ```

---

## 프로덕션 빌드

### Windows 릴리스 빌드

```bash
# Next.js 빌드 (Release 모드)
npm run build

# Tauri 앱 빌드
npm run tauri:build
```

빌드 결과물은 `src-tauri/target/release/bundle/` 디렉토리에 생성됩니다.

- `msi/`: Windows 설치 패키지
- `nsis/`: NSIS 설치 프로그램

### 빌드 옵션

| 모드 | 명령 | 설명 |
|------|------|------|
| 개발 | `npm run tauri:dev` | Hot reload 활성화 |
| 프로덕션 | `npm run tauri:build` | 최적화된 빌드 |
